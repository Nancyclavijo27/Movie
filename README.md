## Documentación API

Puedes acceder a la documentación de la API utilizando Swagger. Sigue estos pasos:

1. Asegúrate de que la aplicación esté en ejecución.
2. Abre tu navegador web y visita: [http://localhost:3001/api-docs](http://localhost:3001/api-docs)  o   https://movapi-0u9d.onrender.com/api-docs/

En la documentación de Swagger, podrás explorar y probar los endpoints de la API, así como obtener información detallada sobre cada uno.

Nota: Asegúrate de tener [Swagger UI](https://swagger.io/tools/swagger-ui/) instalado y habilitado en tu aplicación para que funcione correctamente.

Eso es todo!
